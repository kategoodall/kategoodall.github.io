CSS Selectors

February 14, 2023

When linking HTML elements into the CSS stylesheet:

Class

  • Broader element; HTML ie: div class="banner" (enclosed in < >)
  • Use period to select in CSS >> .banner

ID

  • More specific, usually just one element; HTML ie: img id="name" src="./picture.png"
  • Use # to select in CSS >> #name

Writing CSS

February 12, 2023

When linking HTML to CSS

  • class="header" >> .header
  • id="cook-time" >> #cook-time
  • ul class="ingredients" >> .ingredients li {
  • p class="time" >> p.time {
  • a href="http link" >> a.external-link { (The anchor element can be found within paragraph element; be specific on which element to select)
  • selecting multiple elements >> h1, p, li {

Changing lists

  • Make the bullets square >> list-style: square
